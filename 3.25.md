## 踩坑记录

1. **Minimax System Message 对接问题**：
   - LangChain4j 发送的 system message 在 MiniMax 模型上表现异常
   - 可能与 MiniMax 的 ChatRequest 格式不兼容
   - 症状：system prompt 未生效或响应异常
   - 状态：未解决，需要进一步排查

2. **Jackson 反序列化需要无参构造器**：
   - Jackson 把 JSON 转成 Java 对象时，需要"先创建空对象，再填值"
   - 只有 `@AllArgsConstructor` 不够，必须加 `@NoArgsConstructor`
